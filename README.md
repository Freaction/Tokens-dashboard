# 🎨 Tokens Dashboard

Tokens Dashboard — это профессиональный инструмент для визуализации, глубокого аудита и управления архитектурой дизайн-токенов. Проект создан для того, чтобы превратить хаос в JSON-файлах в прозрачную и управляемую систему.

---

## 🚀 Основные возможности

### 📊 Продвинутая аналитика и метрики

В проект встроена мощная система анализа, которая позволяет не просто смотреть на токены, а понимать состояние вашей дизайн-системы:

#### 🔍 Глубокий аудит (Core Metrics)
- **Usage Metrics:** Анализ используемости токенов — какие реально работают, а какие являются лишними.
- **Alias Depth Analysis:** Контроль цепочек алиасов. Dashboard подсвечивает слишком длинные связи, которые усложняют поддержку.
- **Error Tracking:** Автоматический поиск битых ссылок (broken references), циклических зависимостей и ошибок в типах данных.

#### 🔄 Сравнение версий (Diff Engine)
- **Semantic Diff:** Сравнение логических значений токенов, а не просто текста в JSON.
- **Metric Tracking:** Мгновенный отчет: сколько токенов добавлено, изменено или удалено между любыми версиями.
- **Visual Validation:** Прямой просмотр изменений цвета, типографики и числовых значений.

#### 🕸 Визуализация связей (Node Graph)
- Интерактивный граф, отображающий токены как узлы. Наглядно показывает "генеалогическое древо" от базового примитива до финального семантического токена.

## 💻 Полная автономность (Zero Configuration)

Это полностью независимая версия. Для её работы **не требуется предварительно установленный Node.js** или другие инструменты — всё необходимое уже находится внутри.

1. **Скачайте:** [Загрузите архив с GitHub](https://github.com/Freaction/Tokens-dashboard/archive/refs/heads/main.zip)
2. **Распакуйте:** Извлеките содержимое архива в любое удобное место.
3. **Запустите:**
   - **Windows:** Просто запустите `run-windows.bat`.
   - **macOS / Linux:** Просто запустите `run-mac-linux.sh`.

Dashboard мгновенно откроется в браузере по адресу `http://localhost:3001`.
