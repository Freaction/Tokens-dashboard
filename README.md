# 🎨 Tokens Dashboard

Tokens Dashboard — это профессиональный инструмент для визуализации, глубокого аудита и управления архитектурой дизайн-токенов. Превратите хаотичные JSON-файлы в структурированную и прозрачную систему с автоматическим контролем качества.

---

## 🏆 Индекс здоровья системы (Health Score)

В Dashboard встроена уникальная система оценки качества вашей дизайн-системы. На основе 100-балльной шкалы приложение анализирует каждый токен и выставляет оценку:

- **90–100 (Excellent):** Идеальная структура, готовая к масштабированию.
- **70–89 (Good):** Хорошая база, требующая небольших оптимизаций.
- **50–69 (Warning):** В архитектуре есть критические ошибки или замусоренность.
- **< 50 (Critical):** Система требует немедленной ревизии.

### 🔍 Что мы анализируем (Key Metrics):
- **Broken References (🚨 Fatal):** Поиск ссылок на несуществующие токены. Это критические ошибки, ломающие сборку.
- **Circular References (🚨 Fatal):** Детекция циклических зависимостей, предотвращающая бесконечную рекурсию.
- **Alias Depth Control:** Анализ глубины вложенности алиасов. Оптимально — до 3 уровней. Большее количество шагов усложняет аудит и поддержку.
- **Unused Primitives Analysis:** Выявление "мертвого кода" в палитрах — токенов, которые не используются ни в одном семантическом значении.
- **Hardcoded Values Check:** Проверка на соответствие принципу "Single Source of Truth". Мы подсветим семантические токены, которые используют прямое значение вместо ссылки на примитив.
- **Cyrillic Characters Detection:** Поиск русских букв в путях токенов, которые могут вызвать сбои в CI/CD инструментах.

## 🔄 Сравнение версий (Semantic Diff)
Dashboard не просто сравнивает текстовые файлы — он понимает структуру токенов:
- **History Tracking:** Сравнивайте изменения между любыми версиями (добавлено, изменено, удалено).
- **Metric Tracking:** Мгновенный отчет об изменениях в реальном времени.
- **Visual Validation:** Прямой просмотр изменений цветов, параметров типографики и числовых значений.

---

## 🕸 Визуализация связей (Node Graph)
Интерактивный граф (Node Canvas) отображает "генеалогическое древо" каждого токена. Вы видите весь путь: от базового примитива до финального семантического токена.

---

## 💻 Полная автономность (Zero Configuration)

Это полностью независимая версия. Для её работы **не требуется предварительно установленный Node.js** или другие инструменты — всё необходимое уже находится внутри (для Windows бинарный файл Node.js скачивается автоматически при сборке).

1. **Скачайте:** [Загрузите архив с GitHub](https://github.com/Freaction/Tokens-dashboard/archive/refs/heads/main.zip)
2. **Распакуйте:** Извлеките содержимое архива в любое удобное место.
3. **Запустите:**
   - **Windows:** Запустите `run-windows.bat`.
   - **macOS / Linux:** Запустите `run-mac-linux.sh`.
